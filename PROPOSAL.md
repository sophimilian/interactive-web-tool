# Mini Project #2 - Flashcard App

## What I'm building
- A web-based flashcard app where you can create, flip through, and manage study cards for any subject. 

## Who it is for/Why
- It is for myself--I want a simple way to make and review flashcards for any one of my classes without needing an account or paying for a subscription.

## Core Features
1. Add a card by typing a question (front) and answer (back)
2. Flip a card by clicking it to reveal the answer
3. Navigate through the deck with Previous / Next buttons
4. Shuffle the deck to randomize study order 
5. Delete cards that you no longer need 
6. Reset back to the first card 

## What I do not know yet
- How to make the 3D flip animation with CSS transforms 
- How to correctly store cards as objects in an array and keep track of which card is showing (with an index)
- How to update the DOM when a card is added/deleted without breaking the current position in the deck
